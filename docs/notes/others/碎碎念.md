## 免费使用校园网流量

### 服务器+winscp

本地通过校园网认证之后，使用内网服务器下载文件，使用winscp从内网服务器传输文件到本地

原理：

1. 使用服务器的校园专线从外部传输数据到服务器（消耗服务器流量）
2. winscp使用ftp内网连接，校园内网传输虽通过校园认证服务器中转但不消耗本地校园网账号流量

操作：

1. 复制链接并把命令输入服务器命令行并回车

```bash
wget --user-agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/58.0.3029.110 Safari/537.36" "在这里输入下载链接"
```

2. 使用winscp连接服务器
