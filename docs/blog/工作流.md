## 写笔记转博客工作流

- 配置Plume主题自动构建markdown为静态网页：https://theme-plume.vuejs.press/

- 配置typora软件

- 使用git上传笔记转为博客

  ```sh
  git add .
  git commit -m "update"
  git push
  ```
